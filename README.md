# WinAPI-Music-Style-Project
This project used to remix popular music into WinAPI style without adding anything else but GCC (c++ complier) and basic lib in windows (it doesnt need any sofware like music player to work too)

How to build main.cpp into .exe files (windows):
- download the source file (main.cpp)
- open cmd
- use cd command to locate to folder contains main.cpp
- use this command to compile: g++ main.cpp -o <music_name> -lwinmm (you need to have mingw installed or just download codeblock+mingw and follow tutorial to download it)
- run the .exe file and here we go!
